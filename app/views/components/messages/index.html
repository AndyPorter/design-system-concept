When to use this component

Use the message panel where the user will be looking - for example at the top of the page, or above where the user focus is. Users find it easy to miss feedback messages, expecially if not in view, eg on a mobile

More research needs to be done, to validate these patterns and to explore other ways of informing the user, such as with a modal message
